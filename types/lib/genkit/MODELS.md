# Gemini Models for Genkit (อัปเดต ธันวาคม 2025)

## โมเดลแนะนำ (เรียงตามประสิทธิภาพ + ความพร้อม)

| Model                    | ความเร็ว | ความฉลาด | ภาษาไทย | ค่าใช้จ่าย | แนะนำสำหรับ          |
|--------------------------|---------|----------|---------|------------|---------------------|
| `gemini-2.5-flash`       | เร็วมาก | สูงมาก   | ดีเยี่ยม | ถูก        | Production (แนะนำ)  |
| `gemini-2.5-pro`           | ปานกลาง | สูงสุด   | ดีที่สุด | แพง        | งานซับซ้อนมาก       |
| `gemini-2.0-flash-exp`   | เร็ว     | สูง      | ดี       | ถูก        | ถ้า 2.5 ยังไม่พร้อม   |
| `gemini-1.5-flash`       | เร็ว     | ดี       | ดี       | ถูกสุด     | Fallback สุดท้าย     |

## วิธีเปลี่ยนโมเดล (แนะนำ)

```ts
// lib/genkit/config.ts
import { gemini25Flash } from '@genkit-ai/googleai';

export const defaultModel = gemini25Flash; // ดีที่สุดในปี 2025
